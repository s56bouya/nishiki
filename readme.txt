== Description ==

Nishiki is a fully responsive theme. Elegance,Refined,Multifunctional.

* Responsive layout.
* Front Page Template.
* Play video with Main Visual(YouTube or mp4).

In your admin panel, go to Appearance -> Customize.
About 80 customization can be done without writing the code.
Gutenberg Block Editor Style Support.

== Installation ==

1. In your admin panel, go to Appearance -> Themes and click the 'Add New' button.
2. Type in Nishiki in the search form and press the 'Enter' key on your keyboard.
3. Click on the 'Activate' button to use your new theme right away.
4. Navigate to Appearance > Customize in your admin panel and customize to taste.
